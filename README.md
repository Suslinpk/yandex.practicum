# yandex.practicum

Этот репозиторий посвящен учебным проектам, которые я выполнил за время прохождения курса «Аналитик Данных» от Яндекс.Практикуме.

Данный курс довольно объемный и он помог мне улучшить навыки обработки и систематизации данных, обнаруживать закономерности, выдвигать гипотизы и четко сообщать важные выводы.
